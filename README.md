# 🚛 Sistema de Gerenciamento de Frota

Projeto desenvolvido em Java utilizando Programação Orientada a Objetos (POO), com foco em controle de veículos e boas práticas de desenvolvimento.

---

## 📌 Funcionalidades

- Cadastro de veículos (Caminhões)
- Cálculo de autonomia
- Verificação de necessidade de abastecimento
- Identificação de revisão urgente
- Seleção de tipo de manutenção

---

## 🧠 Conceitos Utilizados

- Encapsulamento
- Herança
- IF / ELSE
- SWITCH
- Vetores (Arrays)
- Laços de repetição (FOR)
- Entrada de dados com Scanner

---

## 🛠️ Tecnologias

- Java
- IntelliJ IDEA
